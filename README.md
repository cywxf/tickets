
＃＃Python 实现火车票查询工具＃＃

The code for python tickets project.

Train tickets query via command-line.

Usge:
    tickets [-gdtkz] <from> <to> <date>

Options:

    -h,--help 显示帮助菜单

    -g        高铁
    -d        动车
    -t        特快
    -k        快速
    -z        直达

Example:

     tickets beijing shanghai 2016-08-25
